# AI行业自媒体个人网站

这是一个以极简画廊编辑风 (Minimalist Editorial) 为基调的个人网站，旨在展示我的 AI 相关文章、摄影集和 AI 视觉作品。

## 项目特点
根据 `.cursorrules` 的指导原则，为了让没有编程基础的用户获得最省力、最简单的操作体验（无需安装 Node.js、npm 等任何环境），本项目采用了纯 **HTML + CSS + JavaScript** 的架构进行构建。

### 技术栈
- **HTML5**: 语意化标签，构建页面骨架
- **Tailwind CSS (CDN)**: 高效的实用类样式框架
- **CSS Variables**: 遵循 `DESIGN.md` 中的规范，统一定义全局色彩与排版
- **GSAP (CDN) & ScrollTrigger**: 实现 L2 级别的流畅页面动效与滚动视差
- **Google Fonts**: 使用 `Newsreader`（衬线体）与 `Inter`（无衬线体）构建优雅排版

## 核心版块
1. **Hero Section (首屏)**：优雅留白的欢迎区。
2. **我的文章 (Articles)**：纯文本的简约目录，带细腻悬停动效。
3. **AI 作品 (AI Works)**：非对称网格展示，具备滚动视差效果。
4. **摄影集 (Photography)**：横向交错排版，打破常规网格。
5. **关于我 (About)**：杂志底页般的个人简介与联系方式。
